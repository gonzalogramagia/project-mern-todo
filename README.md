# MERN Task Management App 🚀

A full-stack Todo application built to showcase modern web development skills using the MERN stack (MongoDB, Express, React, Node.js). 

This project was developed as part of a [FreeCodeCamp](https://www.freecodecamp.org/) course to master React, state management, and full-stack integration, then enhanced with custom CI/CD workflows and multi-platform deployment configurations.

## ✨ Key Features

- **Full CRUD Operations**: Create, read, update, and delete tasks seamlessly.
- **Dynamic UI**: Built with **Chakra UI** for a polished, responsive, and accessible user experience.
- **Dark/Light Mode**: Full support for theme toggling with persistent user preference.
- **State Management**: Orchestrated with **Zustand** for lightweight and efficient global state.
- **Real-time Feedback**: Interactive toast notifications for user actions and error handling.
- **Modern Routing**: Clean client-side navigation using **React Router 7**.

## 🛠️ Tech Stack

**Frontend:**
- [React](https://reactjs.org/) (Version 18)
- [Vite](https://vitejs.dev/) (Fast Build Tool)
- [Chakra UI](https://chakra-ui.com/) (Component Library)
- [Zustand](https://github.com/pmndrs/zustand) (State Management)
- [React Router 7](https://reactrouter.com/) (Routing)

**Backend:**
- [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/) (API Layer)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) (Cloud Database)
- [Mongoose](https://mongoosejs.com/) (ODM)
- [Dotenv](https://www.npmjs.com/package/dotenv) (Environment Management)

## 🌐 Deployment

- **Frontend:** Hosted on [GitHub Pages](https://pages.github.com/).
- **Backend:** Hosted on [Render](https://render.com/).
- **CI/CD:** Automated deployments using GitHub Actions.
